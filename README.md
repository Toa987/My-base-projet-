# My Base Project

Petit projet d'exemple pour Base (réseau EVM).
Contenu :
- `contracts/SimpleStorage.sol` : contrat Solidity minimal.
- `index.html` : page web simple pour se connecter au wallet et lire/écrire une valeur.

Instructions rapides :
1. Forkez ce dépôt (bouton **Fork** en haut à droite).
2. Ouvrez `index.html` dans un serveur local (ou hébergez-le) et connectez votre wallet (Coinbase Wallet / MetaMask).
3. Pour déployer le contrat utilisez Remix ou Hardhat et choisissez le réseau Base (testnet).
